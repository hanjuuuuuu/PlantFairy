# 식물요정

## OpenAI를 이용한 식물 추천 및 관리 웹 사이트  
 - 사용자 기호 별 식물을 추천해주고, 식물 키우기에 도움이 되는 관리방법을 제시하는  프로젝트 :eyes:
   <br></br>
### 2023 한국인터넷정보학회 추계학술발표대회 학술지 게재 및 포스터 논문 발표 :memo:

### Front - End
 - `React`  
 - `css`
 - `scss`

### Back - End
 - `Node.js`
 - `Express.js`
 - `MySQL`

### OpenAI
- `text-davinchi-003`
- `chatGPT`
- `DALLE2`
   
### <주요 기능>
- 식물 추천 기능
- To-do list
- 회원가입 및 로그인
- 커뮤니티
- 포인트 적립

- 레벨 기능
